Setup
=====
To setup to library you need at least Python 3.6 and pip installed.
.. code-block:: bash
    $ pip install ai-katie

To build it directly from the code:
.. code-block:: bash
    $ pip install -r requirements.txt
    $ pip install .
    $ pytest
